Los archivos de sonido para el juego Simon Dice deben ser archivos MP3 con los siguientes nombres:

1. green.mp3 - Un tono agudo para el botón verde (Do/C4 - 261.63 Hz)
2. red.mp3 - Un tono medio-agudo para el botón rojo (Mi/E4 - 329.63 Hz)
3. yellow.mp3 - Un tono medio para el botón amarillo (Sol/G4 - 392.00 Hz)
4. blue.mp3 - Un tono más grave para el botón azul (Do/C5 - 523.25 Hz)
5. wrong.mp3 - Un sonido de error para cuando el jugador se equivoca

Nota: Si los archivos de sonido no están disponibles, el juego utilizará la API Web Audio para generar tonos simples.
